1. Introduction and History

Duration: 15-20 minutes

What is Version Control?

Version control is a system that records changes to files over time so you can recall specific versions later. Think of it as a "time machine" for your code!

Local VCS

Keep versions on your local machine only

Centralized VCS

One central server holds all versions (SVN, CVS)

Distributed VCS

Every clone is a full backup (Git, Mercurial)

History of Git

Created by Linus Torvalds in 2005 for Linux kernel development. Born out of frustration with existing tools!
